# 營隊七堂課總整理｜教材・流程・App 備忘

對象：國中為主＋少數高中｜設備：手機＋投影＋穩定 Wi-Fi｜繳交管道：**Padlet**（每堂一列，課尾固定儀式「上傳 Padlet」）

| 2 | 7/7（二）14:00 | 角色貼圖工坊 | 8 張去背貼圖 PNG |

順序判斷：這樣排是對的，不用動。理由——Canva 遊戲課放午後第一節（最想睡的時段給最好玩的內容），且它天然是「補進度緩衝」，卡在貼圖或影片的學生這節補完，第 6 堂人人有素材；網站(上)(下)隔一夜，剛好讓你晚上巡檢每個人的素材缺口。

## 三個已定案的關鍵決策

1. **主題曲可下載**：生成時**選較舊模型**即可免費下載 MP3（已實測；開營早上再驗一次，平台隨時可能改規則）。MP3 當第 4 堂影片配樂、分享連結嵌網站；下載臨時失效就退回「連結＋音樂庫」方案。
2. **影片雙軌制**：底線版＝Gemini 生 3 張場景圖（附參考圖，複習一致性三招）＋Canva 動態效果＋字幕＋主題曲 MP3，人人達標；進階版＝挑 1–2 個鏡頭用 Canva「圖生影片」做動態片段（**額度有限——今晚實測額度、秒數、生成等待時間**）。剪輯全程 Canva，不另教 CapCut／LightCut。
3. **貼圖不承諾 LINE 上架**（審核要數小時～2 天）：產出定義為圖檔，放網站展示。想上架的回家由家長協助。

## 第 1 堂｜AI 溝通術（7/7 13:00）

- 產出：角色定裝照＝之後所有課的參考圖
- 流程：外星實習生開場 5' → 實驗「畫一隻貓」10' → 五指口訣拆解 7' → 手寫 prompt 實作＋迭代 23' → 定裝＋存圖＋上傳 Padlet 10' → 分享 5'
- 教材：`prompt-teaching-slides.html`（頁 1–9）、`prompt-lesson-plan.md`、救援用 `creative-character-lab-course-v6.html`
- App：Canva Magic Media（短 prompt 實驗）、Gemini（長 prompt＋定裝照）
- 備案：預生成三張貓（壞／中／好）＋一張定裝照範例

## 第 2 堂｜角色貼圖工坊（7/7 14:00）

- 產出：8 張去背貼圖 PNG
- 流程：AI 失憶實驗 8' → 一致性三招 7' → 8 格連發生成 25' → Canva 去背裁切 12' → 第 9 張挑戰（徒手驗收）6' → 收尾 2'
- 教材：`prompt-teaching-slides.html`（頁 10–14）、v6 工具（完整版貼圖 prompt）
- App：Gemini（附參考圖生成）、Canva（去背＝教育版功能）
- 備案：整張 8 格失敗 → 一次生 1–2 格；預做一組貼圖示範

## 第 3 堂｜AI 個人主題曲（7/8 10:00）

- 產出：主題曲一首，分享連結上傳 Padlet
- 流程：聽 2 首範例（好壞歌詞對比）8' → 歌詞產生器寫詞（Gemini）15' → Suno 生成＋選版本 25' → 下載 MP3＋貼連結＋互聽 12'
- App：Gemini（寫詞）、Suno（免費版每日 50 credits ≈ 10 首；**選較舊模型可下載 MP3**）
- 提醒：Suno 要帳號——行前通知準備 Google 帳號；一人生 2–3 版就好，別燒額度
- 教材：`music-video-slides.html`（頁 1–6，第 3 頁是互動歌詞產生器）＋ v6 工具「主題曲歌詞」分頁（自動帶入角色資料）

## 第 4 堂｜30 秒角色宣傳片（7/8 11:00）

- 產出：30 秒 MP4，上傳 Padlet
- 流程：範例＋三段式腳本 8' → Gemini 生 3 張場景圖（附參考圖）12' → Canva 組裝＋動態效果 15' → 字幕＋主題曲 MP3 10' → 匯出上傳 10' → 快閃放映 5'
- 教材：`music-video-slides.html`（頁 7–13，含分鏡表、填空腳本、場景圖 prompt 範本）＋ `character-camera.html`（角色合成相機：相機實拍疊 PNG／3D 模型，拍照內建；PNG 模式可網頁錄影，3D 模式用螢幕錄影）
- 3D 進階：想玩 3D 的學生先把角色圖轉 .glb（你已實測可行），直接餵給合成相機；轉檔失敗就用去背 PNG，一樣能拍
- App：Canva 影片編輯（全程）
- 提醒：全班同時匯出會塞——分批；30 秒內文字自己上，不依賴自動字幕
- 備案：動畫做不來 → 圖片輪播＋文字就已達標

## 第 5 堂｜Canva AI 玩具工坊（7/8 13:00）

- 定位：放鬆＋自由創作＋**補進度緩衝**（貼圖、主題曲、影片沒完成的在這節補）
- 流程：示範 1–2 個 Canva AI 好玩應用 10' → 自由發想製作 35' → 互相玩＋分享 15'
- 教材：`website-slides.html`（頁 1–2）
- App：Canva（AI 生成、互動內容功能）
- 提醒：這節唯一「必須」完成的事：每個人 Padlet 上四樣素材（定裝照、貼圖、歌連結、影片）到齊——網站課的入場券

## 第 6 堂｜個人網站架設（上）（7/8 14:00）

- 產出：網站初版（結構完整、素材就位）
- 流程：看完成範例網站 8' → Canva 網站模板＋四區結構（角色介紹／貼圖牆／主題曲／宣傳片）12' → 從 Padlet 取素材填入 35' → 存檔 5'
- 教材：`website-slides.html`（頁 3–6，網站四區結構）
- App：Canva 網站
- 提醒：主題曲用「連結按鈕」嵌 Suno；影片直接上傳 MP4
- 你今晚的作業：先做一個範例網站當 demo

## 第 7 堂｜個人網站架設（下）＋發布（7/9 10:00）

- 產出：公開網站＋QR code
- 流程：內容補完＋排版微調 25' → 發布（canva.site 網域）＋產 QR code 10' → 掃碼互訪＋留言互評 20' → 結業合影（全班 QR 牆）5'
- 教材：`website-slides.html`（頁 7–12，含品牌一致性檢查＋個資安全）
- App：Canva 網站發布（內建 QR）
- 提醒（個資）：網站用暱稱、不放學校班級與真實全名；用真臉照片的學生確認家長同意書涵蓋
