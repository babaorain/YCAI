# 第 1–2 堂教案｜AI 溝通術：從 Prompt 到貼圖

對象：國中為主、少數高中生｜設備：手機＋投影｜每堂 60 分鐘

## 核心設計理念

先徒手「有感」→ 再拆架構 → 實作迭代 → 徒手驗收。
prompt 生成器（v6 工具）是**救援用**，不是主線——落後或卡住的學生才開給他用。學生自己寫得出 prompt，第 3–7 堂（歌詞、影片、網站）才帶得動。

## 工具分工

| 工具 | 用途 | 原因 |
|---|---|---|
| Canva Magic Media | 第 1 堂前半的快速實驗 | 學生已在課堂內、prompt 短、速度快 |
| Gemini | 角色定裝照、貼圖連發、所有需要一致性的生成 | 吃得下長 prompt、可附參考圖 |
| 投影教材 | prompt-teaching-slides.html | 對比實驗、五指口訣、任務卡都在裡面，示範 prompt 有複製鍵 |
| v6 工具 | creative-character-lab-course-v6.html | 備案／救援／快的學生自由探索 |

## 五指口訣（全課程共用的 prompt 架構）

| 手指 | 元件 | 問自己 | 範例 |
|---|---|---|---|
| 👍 拇指 | 主角 | 誰？ | 一隻橘色胖貓 |
| ☝️ 食指 | 特徵 | 長什麼樣？（固定點） | 圓框眼鏡、一臉不爽 |
| 🖕 中指 | 畫風 | 什麼風格？ | 柔和 3D 動畫風格 |
| 💍 無名指 | 格式 | 要什麼輸出？ | 全身、白色背景 |
| 🤙 小指 | 禁止 | 不要什麼？ | 不要文字、不要浮水印 |

## 第 1 堂（7/7 13:00–14:00）AI 溝通術：Prompt 與角色設計

產出目標：一張「角色定裝照」（白背景全身圖）＝之後所有課的角色參考圖。

| 時間 | 環節 | 做什麼 |
|---|---|---|
| 00–05 | 開場鉤子 | 「AI 是一個很會畫圖、但**完全不認識你**的外星實習生。你沒說的，它全用猜的。」 |
| 05–15 | 實驗①：畫一隻貓 | 全班在 Canva 輸入同樣三個字「畫一隻貓」→ 跟旁邊同學比：一樣嗎？為什麼？（每個人的貓都不同＝AI 在猜） |
| 15–22 | 五指拆解 | 投影升級版 prompt（加特徵 → 加畫風格式禁止），五種顏色標出五個元件，帶出五指口訣 |
| 22–45 | 實作：寫自己的角色 | 學生用五指架構**手寫**自己角色的 prompt → Gemini 生成 → 不滿意時**只改一個元件**再生（迭代法） |
| 45–55 | 角色定裝 | 選出最滿意的一張＝「角色參考圖」→ 存到相簿＋上傳作品區（沒做這步第 2 堂會塞車） |
| 55–60 | 分享＋預告 | 抽 2–3 人投影分享 prompt＋成品；預告：「明天同一個 prompt 再生一次，會發生怪事」 |

講稿關鍵句：

- 「你不是在畫圖，你是在**下訂單**。訂單寫得越清楚，出貨越準。」
- 「不滿意的時候，不要整段重寫——**一次只改一個地方**，你才知道是哪裡造成差別。這叫科學家精神。」

## 第 2 堂（7/7 14:00–15:00）角色貼圖工坊

產出目標：8 張去背貼圖圖檔（存相簿＋上傳作品區）。

| 時間 | 環節 | 做什麼 |
|---|---|---|
| 00–08 | 實驗②：AI 失憶 | 把第 1 堂的 prompt 原封不動再生一次 → 跟定裝照比 → 變了！帶出：**AI 沒有記憶** |
| 08–15 | 一致性三招 | ①附上角色參考圖 ②把固定點寫死（臉、髮、服裝、配色「完全相同」）③只改表情動作 |
| 15–40 | 8 格貼圖生成 | Gemini 附參考圖＋8 格 prompt（投影教材有短版可複製；v6 工具有完整版）。失敗的學生改成一次 1–2 格 |
| 40–52 | Canva 去背裁切 | 把 8 格大圖上傳 Canva → 去背 → 逐格裁切 → 下載 PNG → 上傳作品區 |
| 52–58 | 第 9 張挑戰（驗收） | **不用任何工具**，手寫第 9 個表情的 prompt（必含五指五元件＋一致性指令）→ 給老師過目才能生成 → 通過＝「貼圖大師認證」 |
| 58–60 | 收尾 | 預告明天主題曲 |

第 9 張挑戰＝這兩堂課的評量。通過標準：prompt 含五元件，且有「同一角色、保持不變」的一致性指令。

## 差異化

- 太快：做第二組表情、用 v6 工具做角色資料卡、幫同學 debug prompt。
- 卡住：開 v6 工具用表單引導；完全沒靈感 → 按「隨機骰一個角色」直接開跑。

## 常見卡點應對

| 狀況 | 應對 |
|---|---|
| 貼圖上的中文字變亂碼 | 預設不加字；要加字改在 Canva 疊字（順便預告第 6 堂） |
| 手、眼睛畫壞 | 在禁止元件加：不要多手指、不要變形 |
| Canva 輸入框字數不夠 | 長 prompt 一律改用 Gemini |
| 全班同時生成很慢 | 分兩批錯開；等待時先寫下一個 prompt |
| 角色還是不像 | 檢查參考圖有沒有真的附上；固定點是否寫「完全相同」 |
